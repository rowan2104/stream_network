First byte: Packet Type byte
bit 0: data or control

bit 8: for controll, yes or no
bit 7: for control, request


BYTE     :  PURPOSE             : SENDERS - RECIEVERS   : ARG BYTES :               : DATA
00000000 : ERROR                : (any-any)             : 1         : error code    :


10000000 : disconnected
10000001 : connected

10000010 : client request disconnect
10000011 : client request connection

10000110 : request unsubscribe
10000111 : request subscribed

1XXX0010 : request active streams
1XXX0011 : list active streams

10000100 : unsubscribe
10000101 : subscribed

10001011 : prod request connect
10001010 : prod request disconnect

10001001 : prod connect
10001000 : prod disconnect

1XXX1110 : request X stream deletion

1XXX0100 : X stream deletion
